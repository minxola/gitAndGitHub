Curso de Git

Que es Git?

Git es un sistema de control de versiones utilizado para la gestion de todos los cambios de proyectos. Generalmente es utilizado en proyectos informáticos, sin embargo también puede ser utilizado en otro tipo de proyectos.

Los cambios que realizamos en Git de manera local, se guardan en una base de datos. Todos estos cambios tambien lo podemos guardar en repositorios remotos, ya sea que sean nuestros o de algún proveedor de este servicio como lo son GitHub, GitLab y GitBucked.
